JavaScript

Object-oriented programming language used to create interactive events within a browser

Modulo

Returns the remainder of dividing two numbers. %

Operators

Symbols used to perform an operation
i.e. + - * /

NaN

Not a Number

Variable Naming Conventions

Variables begin with a lower-case letter and each following word is capitolized

+=

Updates the variable with the addition provided.

Function

Slice of code to perform a particular task.

Method

Interface to access and modify the data properties of an object

Argument to a method

Used when calling a method to perform a specific task on it

String

Any sequence of characters ensconced in quotation marks

Chaining methods

Calling one method after another on an object in one continuous line

Escape

The character "/" turns special characters to string characters

Alert

Pops up a dialog box in the browser with a message

Commenting code

Allows a programmer to add lines that the program will skip

Booleans

Either true or false, not strings

Undefined

Property of variable that has not been assigned a value

Parameter

The names listed in a function definition
i.e. function(parameter1, parameter2)

Return

Stops execution of a function and returns a value
